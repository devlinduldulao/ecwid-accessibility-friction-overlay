# Marketplace Assets

These files are the marketplace-facing assets for the Ecwid app listing.

Included:

- `app-icon.svg` and `app-icon.png` for the app icon
- `listing-banner.svg` and `listing-banner.png` for the marketplace banner
- `control-room-idle.png` captured from the live merchant dashboard in its idle state
- `control-room-preview.png` captured from the live merchant dashboard with preview mode enabled

Before submission, export any extra marketplace-specific raster sizes required by the target store and review the screenshots after major UI changes.
