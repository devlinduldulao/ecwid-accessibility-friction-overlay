This Ecwid version is designed as a merchant-facing control room. No database, no Redis, and no long-running Node server are required in production. The admin page is the main product surface for business owners, while the storefront script is a lightweight collector used for merchant-run QA walkthroughs and preview drills.

Deployment Model

Store: demo-store

Runtime: Static HTML, JS, and CSS on any HTTPS host.

Tradeoff: No cross-device live dashboard stream without a backend.

Storefront Tracking Controls

Deploy

Paste-Ready Ecwid Loader

Paste this into Ecwid Control Panel → Design → Custom JavaScript. It installs the owner-configured collector, keeps customer-facing UI hidden by default, and loads assets from your static host.
