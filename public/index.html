Getting Started — How This App Works

Three steps to start seeing real accessibility friction on your storefront.

1

Configure & Deploy

Adjust tracking settings below, then copy the generated code snippet. Paste it into Ecwid Admin → Website → Design → Custom JavaScript. That's the only setup needed.

2

Open Your Storefront

Visit your live store with your private debug token appended to the URL (shown in the Deploy section below). The debug overlay appears only for you — shoppers never see it.

3

Browse & See Live Data

Real metrics appear in the storefront overlay as you browse. Tab through pages, click around, test forms — the overlay shows live friction events, hotspots, and recommendations in real time.

Important: This admin page shows only demo/preview data for testing layouts. Your real accessibility metrics are always in the storefront debug overlay on your live store — that is where you review actual friction data. No backend or paid service is needed.

Where Do I Find My Real Metrics?
